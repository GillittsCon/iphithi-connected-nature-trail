Project: iPhithi Nature Reserve QR Trail

Goal:
Build a static mobile-friendly interpretive trail app for hackathon demo.

Tech:
React + Vite
Static site
GitHub Pages friendly
No backend
No database
No auth
No paid APIs

Demo path:
Home → Stop → Toggle Quick/Deeper

Stops:
1. Pollinators / Grassland
2. Water / Wetland
3. Green Corridors / Woodland

Content rules:
- Each stop starts with a Before image
- No Now image because visitor is standing in the now
- Each stop includes Quick View and Go Deeper
- Each stop must connect reserve → suburb → garden
- Include small Try this at home action

Data/location plan:
- content in src/data/markers.json
- images in /public/images/

Priority:
Strong, reliable hackathon demo with low risk and simple structure.

Skills to use:
- vite-static-react-builder
- outdoor-mobile-ui
- structured-content-loader
- hackathon-demo-ux

Important rule from skills:
- Demo path in 3 clicks: Home → Stop → Toggle Quick/Deeper
